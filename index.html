Flex-elements flex-grow - определяет пропорции увеличения элемента по сравнению с другими, по умолчанию 0 - размер элемента не меняется, в зависимости от размера экрана



Flex-elements flex-shrink - определяет пропорции уменьшения элемента по сравнению с другими, по умолчанию 1 - размер элемента не меняется, в зависимости от размера экрана, при уменьшении окна сначала уменьшаться элементы с наибольшим flex-shrink, затем следующие элементы с наибольшим flex-shrink, не считая предыдущего и т.д.



Flex-elements flex-basis - определяет размер элемента по умолчанию, auto - размер содержания элемента



Flex-elements flex: [flex-grow] [flex-basis = 1] [flex-basis = auto]

Также пройдет комбинация параметров [flex-grow] [flex-basis] - браузер поймет, что это базиз по px
